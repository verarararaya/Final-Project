N-to-1 Neural Network

Artificial Neural Network is a machine-learning algorithm. It was designed to modify human brain’s construction and try to solve problem as human. A classic ANN model can be described using nodes and connections. There are three layers of nodes, or ‘neurons’, input, hidden, and output. Connection between neurons cover all possible paths from any input nodes to any output nodes. ANN model tries to ‘understand’ the input data and generate neurons to place in the model. Its training process is rather complicated. ANN was popular before SVM became the common choices. It got out fashion as its complexity and high requirement for computing power. Most existing peptide predictors are constructed using SVM. These years, ANN come back with more attention. We think applying ANN to anti-cancer peptide predictor would be an innovating experiment. The training neural network used in this project is called N1-NN, N-to-1 Neural Network. This modelchooses fixed features to compress instead of all information like most models do. After beforehand data processing, the data will be fed forward through neural network and trained by the back-propagation algorithm. To prevent overparametrization, the model can be controlled by these parameters: NHF(the number of units in the hidden layer of the sequence-to-feature network), NHY(the number of hidden units in the feature-to-output network), NF(the number of hidden states in the feature vector f), context(semi-window which is determined by the output net) and epsilon(learning rate).
